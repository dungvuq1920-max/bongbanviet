# Credentials Manifest

This folder is for metadata only.

`manifest.json` can be generated by `npm run sync:from-n8n` or `npm run sync:watch`. It stores credential `id`, `name`, and `type` so workflow credential bindings edited in n8n UI can be reviewed in Git.

Secret values are never exported. Keep real API keys in n8n Credentials, Railway variables, or `.env` files that are not committed.
