Đăng nhập Douyin

Dán cookie từ trình duyệt để xác thực tài khoản

Cách lấy cookie: 1. Mở douyin.com → đăng nhập tài khoản
2. Nhấn F12 → tab Application → Cookies → https://www.douyin.com
3. Tìm dòng sessionid → click chuột phải → Copy all as header value
Hoặc dùng extension Cookie Editor → Export → Paste bên dưới
