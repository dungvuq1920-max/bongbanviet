Hướng dẫn auto post lên Facebook Page theo lịch

  1. Vào Auto Schedule, nhập Page ID và Page Access Token có quyền pages_manage_posts, rồi bấm Lưu cấu hình.
  2. Bấm Kiểm tra token. Khi Page và Token đều báo sẵn sàng, bật Auto Schedule.
  3. Tạo bài bằng Copy Master Prompt hoặc nhập JSON. Mỗi bài cần có scheduled_time cách hiện tại ít nhất 11 phút.
  4. Mở từng bài, copy Image Prompt để tạo ảnh, chọn file ảnh vào bài. Khi upload Facebook, hệ thống gửi cả nội dung và ảnh.
  5. Bấm Upload FB để đăng ngay 1 bài đủ điều kiện, hoặc để Auto Schedule tự chạy nền mỗi 2 phút.

Cấu hình Facebook Page

Cách lấy Page Access Token:
1. Vào Facebook Developers → Graph API Explorer
2. Chọn App quản lý Page → Generate Access Token
3. Tick quyền pages_manage_posts, pages_read_engagement, pages_show_list
4. Gọi GET /me/accounts → copy access_token của đúng Page
5. Dán Page ID và Page Access Token vào form dưới đây
Cach don gian nhat: dan User Access Token co quyen quan ly Page vao o token, bam Ket noi Page tu token. He thong se tu goi /me/accounts de lay Page token. Neu hien nhieu Page, bam Chon ID roi bam ket noi lai. Mo nhanh: Graph API Explorer.
